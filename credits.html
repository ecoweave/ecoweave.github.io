This site is built using a number of open source projects and free tools. They're great, give them a try.

Bootstrap

Bootstrap is a free collection of tools for creating websites and web applications. It is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. It contains HTML and CSS-based templates for typography, forms, buttons, navigation and other interface components.

Glyphicons

GLYPHICONS is a library of precisely prepared monochromatic icons and symbols created by freelance graphic and web designer Jan Kovařík. The icons have been created with an emphasis on simplicity and ease of orientation. They are great for websites, web applications, infographics, wireframes and printing!

Jekyll

Jekyll is a simple static site generator. It takes a template directory containing raw text files in various formats, runs it through Markdown and Liquid converters, and produces a complete, ready-to-publish static website. Jekyll is a free-software, written in Ruby by Tom Preston-Werner, GitHub's co-founder. Jekyll is the engine behind GitHub Pages, a GitHub feature that allows users to host websites based on their GitHub repositories.

GitHub Pages

GitHub Pages are public webpages freely hosted directly from your GitHub repository and easily published through GitHub. GitHub Pages support Jekyll in addition to supporting regular HTML content.

Lightbox for Bootstrap

A lightbox module for Bootstrap that supports images, YouTube videos, and galleries - built around Bootstrap's Modal plugin.
